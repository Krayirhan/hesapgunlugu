This document is a high-level overview. For setup, contribution, and architecture,
use the single source of truth:

## Highlights

- Income/expense tracking with categories and budgets
- Scheduled payments and recurring transactions
- Statistics, charts, and filtering
- App lock (PIN + biometrics)
- Local encrypted backup and restore
- Accessibility-focused UI

## Build Variants

- Free: transaction limits, ads placeholder
- Premium: no limits, no ads
